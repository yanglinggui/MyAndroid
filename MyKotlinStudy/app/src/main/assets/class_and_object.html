类定义

Kotlin类包含：构造函数和初始化代码块、函数、属性、内部类、对象声明等。

使用关键字 class 声明类，后面紧跟类名，如下

class ClassName {
    // 属性
    // 成员函数
    // 初始化代码段
    ... .. ...
}

主构造函数

类可以有零个或一个主构造器，用constructor关键字修饰，主构造器中不能包含任何代码，初始化代码可以放在由init修饰的初始化代码段中。 主构造器的参数可以在初始化代码段中使用,主构造函数参数被var或val修饰时会被默认为类的属性。

主构造函数声明

class ClassName constructor(val firstName: String) {}

如果主构造函数没有任何注解或者指示符，constructor关键字可以被省略

class ClassName(val firstName: String) {}

如果主构造函数注解或者可见的指示符,constructor是必须的

class ClassName public @Inject constructor(val firstName: String) {}

次构造函数

次构造函数也是constructor修饰，如果类有主构造函数，每个次构造函数都要或直接或间接通过另一个次构造函数代理主构造函数。在同一个类中代理另一个构造函数使用this关键字:

class Person(val name: String) {
    constructor (name: String, age:Int) : this(name) {
        // 初始化...
    }
}
